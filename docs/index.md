A RemoteData type for [fp-ts](https://github.com/gcanti/fp-ts)

# Installation

**NPM :**

```sh
npm i fp-ts-remote-data
```

**Yarn :**

```sh
yarn add fp-ts-remote-data
```

`fp-ts-remote-data` versions follow `fp-ts` major versions.

**Peer Dependencies**

| Library | Version  |
| ------- | -------- |
| fp-ts   | `^2.9.0` |
